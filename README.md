# Sistema de Gestão de Documentos

Sistema web desenvolvido em PHP e MySQL para gerenciamento de clientes, usuários, setores, tipos de documentos e uploads.

> Este repositório é uma versão demonstrativa para portfólio, sem dados reais, sem documentos internos, sem arquivos de ambiente e sem informações sensíveis.

## Tecnologias utilizadas

- PHP
- MySQL
- JavaScript
- HTML5
- CSS3
- MVC
- Autenticação
- Upload de arquivos

## Funcionalidades

- Login de usuários
- Controle de acessos
- Cadastro de clientes
- Cadastro de setores
- Cadastro de tipos de documentos
- Gestão de documentos
- Área administrativa
- Perfil do usuário
- Notificações
- Área de envio e consulta de documentos

## Minha participação

Atuei no desenvolvimento da estrutura web com PHP, organização das telas, integração com banco de dados MySQL, regras de autenticação e funcionalidades administrativas.

## Como configurar localmente

1. Clone o repositório.
2. Copie o arquivo `.env.example` para `.env`.
3. Configure os dados do banco MySQL no `.env`.
4. Aponte o servidor local para a pasta `public`.
5. Acesse a aplicação pelo navegador.

## Segurança

O arquivo `.env`, uploads reais, arquivos compactados e dados sensíveis não devem ser enviados para o GitHub.
